# Notes

## SEM_ID Convention

| Suffix | Semester        | Example  |
|--------|-----------------|----------|
| `01`   | Summer Semester | `202501` |
| `02`   | Winter Semester | `202502` |

Find your `SEM_ID` in the URL after clicking **Start** on the Faculty Evaluation page:
`https://iems.bauet.ac.bd/Student/FacultyEvaluation/FacultyList?semId=202502`

## RADIO_VALUE Reference

| Value | Meaning                      |
|-------|------------------------------|
| `1`   | Strongly Agree / Outstanding |
| `2`   | Agree / Very Good            |
| `3`   | Neutral / Good               |
| `4`   | Disagree / Poor              |
| `5`   | Strongly Disagree / Very Poor|

## DELAY_MS Guidance

| Connection  | Recommended `DELAY_MS` |
|-------------|------------------------|
| Fast (fiber)| `300`                  |
| Normal      | `500` (default)        |
| Slow / VPN  | `800–1200`             |
